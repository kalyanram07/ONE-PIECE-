# ONE PIECE: The Grand Line Experience

A premium, cinematic frontend experience showcasing the journey of the Straw Hat Pirates. Built as a demonstration of modern web technologies, GPU-accelerated animations, and immersive design principles.

## 🌟 Features

*   **Cinematic Intro Sequence**: Orchestrated 3-phase loading experience introducing the world.
*   **Immersive Scroll Mechanics**: Powered by Lenis for buttery-smooth scrolling.
*   **Scroll-Driven Animations**: Complex scroll-linked animations using Framer Motion (parallax, opacity fades, scale transforms).
*   **Scroll-Snap Character Showcase**: Full-screen character cards with animated stat bars and 8k AI-generated visuals.
*   **Responsive Glassmorphism Navigation**: Floating desktop navigation and bottom tab mobile navigation.
*   **Modern Stack**: React 19, Vite, TypeScript, and Tailwind CSS v4.

## 🚀 Technology Stack

*   **Framework**: React 19 via Vite
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS v4
*   **Animation**: Framer Motion
*   **Smooth Scroll**: Lenis
*   **Icons**: Lucide React & React Icons
*   **Typography**: Google Fonts (Cinzel, Inter, Noto Sans JP)

## 📦 Installation

```bash
# Clone the repository
# Navigate to the project directory
cd ONE-PIECE

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

## 🎨 Design Philosophy

This project moves away from generic anime website templates and focuses on a **premium, cinematic feel**. It utilizes a dark, immersive color palette, elegant typography (`Cinzel` for headings), and micro-animations to create a feeling of grandeur befitting the Pirate King.
